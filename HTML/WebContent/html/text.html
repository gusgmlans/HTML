HTML 텍스트 서식관련 태그

목차

1. html 서식태그

html에서는 <b>, <i> 등의 태그를 이용하여 굵게, 이탈릭체 등으로 출력한다.

2. 문자 포맷용 태그

태그에 대한 자세한 설명과 예제는 태그 사전이나 태그 레퍼런스 사이트를 참조
추천 사이트 http://www.blognews.co.kr/htmltag/
  • <b> - 글자 강조
  • <em> - 글장 강조
  • <i> - 이탈릭체
  • <small> - 작은 글자
  • <strong> - 글자 강조
  • <sub> - 첨자 글자
  • <sup> - 슈퍼 스크립트
  • <ins> - 삽입 글자
  • <del> - 삭제된 글자
  • <mark> - 마크 글자

3. 컴퓨터 출력용 태그

태그에 대한 자세한 설명과 예제는 태그 사전이나 태그 레퍼런스 사이트를 참조
추천 사이트 http://www.blognews.co.kr/htmltag/
  • <code> - 코드 예제
  • <kbd> - 키보드 입력
  • <samp> - 샘플 글자
  • <var> - 변수 글자
  • <pre> - 눈에 보이는 서식 형태 글자

4. 인용, 정의,표현 등의 태그

태그에 대한 자세한 설명과 예제는 태그 사전이나 태그 레퍼런스 사이트를 참조
추천 사이트 http://www.blognews.co.kr/htmltag/
  • <abbr> - 약어
  • <address> - 주소
  • <bdo> - 글자 진행방향
  • <blockquote> - 들여쓰기
  • <q> - 인용부호
  • <cite> - 인용
  • <dfn> - 용어정의

5. 루비 태그

  • <ruby> - 루비 문자
  • <rt> - 위에 위치하는 문자
  • <rp> - ruby 태그를 지원하지 않을 경우 출력될 문자 표시
